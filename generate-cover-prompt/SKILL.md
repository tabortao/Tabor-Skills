# 生成封面提示词

用于生成公众号文章封面图片的 AI 提示词，生成的图片将直接作为文章封面使用。

---

## 工作流程

### 第一步：信息收集

在生成提示词前，你必须先收集以下信息：

1. **文章标题** - 用户提供的标题
2. **文章内容** - 从上下文中智能提取文章正文（如果存在）
3. **文章类型** - 教程/工具介绍/案例分析/观点分享/资源汇总等
4. **目标受众** - 文章面向的读者群体

### 第二步：内容分析（关键步骤）

**如果上下文中包含文章内容**，你必须：

- 提取文章的核心主题和主旨
- 识别文章的关键价值点和亮点
- 分析文章的情绪基调和风格（专业/轻松/紧迫/温暖等）
- 找出最能代表文章内容的**视觉关键词**
- 确定适合的可视化元素（图标、场景、人物、抽象概念等）

**如果上下文中没有文章内容**，你需要：

- 基于标题本身进行分析
- 明确告知用户："未检测到文章内容，分析基于标题本身。如能提供文章内容，可生成更契合主题的封面。"

### 第三步：封面文字提取

基于标题和文章内容，提取**核心价值/用户收益**，遵循以下优先级：

| 优先级 | 内容类型          | 示例                            |
| ------ | ----------------- | ------------------------------- |
| 1      | **痛点/解决方案** | "告别复制粘贴"、"Markdown 秒发" |
| 2      | **核心功能**      | "自动同步"、"一键发布"          |
| 3      | **用户收益**      | "效率翻倍"、"10 分钟 →1 分钟"   |
| 4      | **情绪共鸣词**    | "绝了"、"神器"、"必看"          |
| 5      | 产品名称          | 放正文介绍，**不要放封面**      |

**提取技巧**：

- 问：用户看完能获得什么？→ 提取这个结果
- 问：解决了什么痛点？→ 提取这个痛点/解决方案
- 问：文章最想传达的情绪是什么？→ 提取情绪关键词
- 产品名、技术细节 → 省略或放正文

### 第四步：生成提示词

使用以下模板结构，根据文章内容定制：

```
一个[风格]的[主体]，[材质/质感]。[视觉细节描述]。
[文字内容]：[提取的封面文字]，字体[字体描述]，[文字效果]。
整体采用[光照/氛围描述]，[色彩方案]，[构图特点]。
图片宽高比:`21:9`, 2K
```

## 特殊规则

1. **文字精简**：封面文字控制在 2-6 个字，最多不超过 8 个字
2. **内容契合**：封面视觉风格必须与文章内容和情绪一致
3. **避免误导**：不要为了视觉效果而使用与内容无关的元素
4. **色彩协调**：根据文章情绪选择主色调（技术-蓝/紫、效率-绿/白、警示-红/橙、温暖-黄/橙）
5. **中文优化**：确保提示词中关于文字的描述适合中文显示
